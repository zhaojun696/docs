    Airtest

    参数含义

    • touch：这个是模拟鼠标左键点击。里面有几个参数

      v：可以是图片或者坐标

      times：默认次数是1，也可以设置成多次

      duration：按住的持续时间，默认是0.01s

      right_click：右键点击（仅限windows模式）

    • wait：这个是等待时间

      v：图片

      timeout：等待多久就会报错，默认值是20s，根据情况适当延长或降低时间

      interval：每次匹配的时间间隔，相当于程序会多久在界面上查询一次

      return：会有返回值，如果找到了就返回图片的坐标，否则抛出一个异常

    • swipe：用来模拟滑动

      x 正值：手指向向右边滑

      x 负值：手指向向左边滑

      y 正值：手指向下边滑滑

      y 负值：手指向向上边滑

      横向移动可以设置 y 的值为0， 纵向移动可以设置 x 的值为 0

    • exits：是否存在想要查找的图片，如果有则返回图片中心坐标点，否则返回false

    • text：用来输入文字内容

    • keyevent：用来模拟一些按键，比如主页、返回等

    • snapshot：用来截图

    • sleep：模拟用户等待时间，默认是1s

    • assert_exits 和assert_not_exits:断言，用来判断这个元素是否存在
